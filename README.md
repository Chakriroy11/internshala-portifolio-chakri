# Chakri Chandaka - Portfolio Website

This repository contains the front-end code for Chakri Chandaka's personal portfolio website. The website is designed to showcase skills, projects, and certifications in a clean, modern, and responsive manner.

## Features

- **Responsive Design:** Adapts to various screen sizes (mobile, tablet, desktop).
- **Dark Mode:** A toggle to switch between light and dark themes for improved user experience.
- **Animated Background:** Subtle animated bubbles create an engaging visual effect.
- **Navigation:** Smooth scrolling navigation to different sections of the portfolio.
- **Home Section:** Introduction with profile image, professional title, years of experience, projects count, and skill percentage. Includes social media links (LinkedIn, GitHub, Instagram) and buttons to download CV and contact.
- **About Me Section:** Detailed description of the developer's background and aspirations.
- **Skills Section:** Visual representation of technical skills (HTML5, CSS3, JavaScript, React.js, Node.js, GitHub, AWS Cloud, MySQL) with progress circles.
- **Certifications Section:** Displays various certifications with images, titles, and links to view certificates.
- **Projects Section:** Showcases different projects with descriptions and links to live demos and GitHub repositories.
- **Contact Section:** Provides contact information (Email, Phone, Location) and a contact form.
- **External Libraries:** Utilizes Tailwind CSS for utility-first styling, Font Awesome for icons, and Google Fonts for typography.

## Technologies Used

- **HTML5:** Structure of the web pages.
- **CSS3:** Custom styling and animations.
- **Tailwind CSS:** A utility-first CSS framework for rapid UI development.
- **JavaScript:** For interactive elements and dark mode functionality (implied, as the provided code only includes HTML and CSS).
- **Font Awesome:** Icon library.
- **Google Fonts:** For the 'Inter' typeface.

## Setup and Usage

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd <project-directory>
    ```
3.  **Open `index.html` in your browser:**
    Simply open the `index.html` file using your preferred web browser. No local server is required as it's a static website.

## Customization

- **Content:** Edit `index.html` to update personal information, project details, skills, and certifications.
- **Styling:** Modify `styles.css` for custom styles or extend Tailwind CSS classes directly in `index.html`.
- **Images:** Replace placeholder images (`./chakri1.jpg`, `aws.png`, etc.) with your own assets. Ensure paths are correctly updated.
- **Links:** Update all social media, project, and certification links to point to your actual profiles and resources.
